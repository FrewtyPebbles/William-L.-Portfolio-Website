# William A. L.'s Portfolio Website

This is my portfolio website where I show live demos of my web-deployable projects along with write ups about all of my projects

# Portfolio Tech Stack

 - Next.js for frontend and backend
 - Docker for portability
 - Terraform with AWS Provider for IaC
 - Packer for Immutable Infrastructure and avoiding VPC Endpoint/NAT gateway costs
 - Prisma for ORM with Postgres for RDBMS
